
📊 Superstore Sales Analysis Dashboard

📌 Project Overview

This project analyzes Superstore sales data using Power BI to uncover key business insights related to sales performance, profit trends, and customer behavior.

🎯 Objectives

- Analyze overall sales and profit performance
- Identify top-performing regions and categories
- Understand customer purchasing patterns
- Provide insights for business growth

🛠️ Tools & Technologies

- Power BI
- Excel / CSV
- DAX

📂 Dataset

The dataset includes:

- Orders data
- Sales and profit information
- Customer and region details
- Product categories

📊 Dashboard Features

- KPIs: Total Sales, Profit, Orders
- Sales analysis by region and category
- Profit trends visualization
- Interactive filters and slicers

🔍 Key Insights

- High sales observed in certain regions but profit is low
- Technology category generates high revenue
- Discounts negatively impact profit
- Customer segments show different buying behavior
